## Usage.
- Clone or Download This Repository in Project Root Dir.
- Initialize package.json
  - -> `yarn init`
- Install All Packages.
  - -> `yarn install`
- When Creating Static Website.
  - -> `gulp`
- When Creating Simple Web Application.
  - Development -> `yarn dev`
  - Build -> `yarn build`
